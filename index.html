What is HTML and CSS?

As a whole, web development is the process of creating and maintaining websites and web applications. Before we dive into the display properties of CSS, we need to know what HTML and CSS are. Web development uses the combination of HTML, CSS, and JavaScript to create websites, style the websites, and give the websites functionality. The process uses HTML to create the framework of the website, CSS to style and shape the website, and it uses JavaScript to dynamically update and change the website.


HTML stands for HyperText Markup Language, and is used as the structure, or the foundations of the website, placing all the items and building blocks on the page. It uses different tags such as <h1>, <p>, <img>, and <div> to display headers, paragraphs, images, and containers. HTML uses these elements to create the skeleton of the website, but it does not dictate the layout or how the content is displayed at all.


CSS stands for Cascading Style Sheets, and is used to control the design and the layout of the website. CSS can be used to change properties on direct elements of HTML or it can use the ids and the classes from HTML. In HTML, an element can be given a class or an Id, and the classes given to elements can also be called in CSS. CSS can be applied to a HTML page in two ways, by embedding the CSS directly in the HTML with a style tag, or by linking an external style sheet. CSS also allows for making the website more appealing and responsive through different screen sizes. JavaScript is used to dynamically change the values in both HTML and CSS, but I will not be going over it on this website.

CSS Layout formatting

When it comes to using CSS to structure the layout of an HTML page, there are three common properties that are extremely useful. These three properties are the display property, the position property, and the margin and padding properties.

The next property is the position property. This property controls how elements are set on the page, with values such as fixed, static, absolute, and relative. The most commonly used are the absolute, fixed, and the relative values, which position the object in the absolute space, fixed on the screen, or relative to other objects respectively. Given the demo above, the four buttons give the box labeled '2' different position values. The first button, position: static;, sets the position to static, which is the default value. When a object is static, it is positioned according to the normal flow of the page.

The next button, position: absolute;, sets the position of the object to an absolute position in its container. This means that the position of the object is dictated by other CSS properties called inset properties. These inset properties come in the form of top, left, bottom, and right, allowing these values to be manually set. The code for the position: absolute; sets top and left to '40px', meaning they are inset by 40 pixels in the container outlined by the orange border. When an object is set to absolute, the other objects nearby do not count it as a physical position, and it is seen that box three takes the spot that box 2 was in due to this.

The third button, position: relative;, sets the position to be relative to the other objects it is around. Using the inset values, an object with the position value of relative can be moved relative to its surrounding objects. In this example, the code gives box 2 an inset left value of '100px', meaning it is set 100 pixels left of its original position.

The last button, position: fixed;, sets the position of the object to a fixed position on the page. Once this is done, the object does not move while the page is scrolled but stays at the location applied to it. Box 2 was given the inset top and left values of '400px', meaning that it is fixed in place at 400 pixels from the top and 400 pixels from the left of the page.

CSS Display Property

The last property is the display property. This property determines how an element behaves in a layout. This property has many uses and can be applied to both the element itself and the children (sub-elements) of an element. The most commonly used values of display are block and inline which determine whether an element will take up the full width of their container or just as much as they need, and the values flex and grid which are extremely useful for setting more complex statements. Given the demo above, there are 5 different display values that can be applied to the container for the three boxes. When the values are applied, they are not applied to the individual boxes but rather the container that the boxes are in.

The first button, display: block;, is a property that is applied to the container itself, not the contents of the container. This property makes the object it is applied to always start a new line and take up the full width available. In the example, it is seen that the container separates itself from the two lines of text and takes the full width available to it.

The second button, display: inline-block;, is an inline property, meaning that the objects it is applied to do not start a new line and only takes up as much space as needed. Since it is also an block property, it can still have width and height values applied to it, however that is not shown in this example. In the example, the container is seen shrinking to the exact size of the three boxes, only taking up the necessary space. It is also seen being in line with the lines of text, not creating line breaks.

The third button, display: flex;, sets the property to flex which is short for flexible. The flexible property allows for a lot of different options, and comes with 7 other CSS properties. The flexbox property is mainly used for 1-dimensional layouts, where there are either rows OR columns, not both. For more information on FlexBox, Click Here.

The fourth button, display: grid;, is similar to the flex property but sets the object to grid instead. The grid layout is a two-dimensional layout that is used to create grids with both rows AND columns. In order to set the rows and columns, the grid layout uses a property call grid-template-columns to set the number of columns and how wide each column is. For more information on CSS grid layout, Click Here.

The final button, display: none;, sets the display property of the object to none. This makes the object not be displayed, and if the object is a container then nothing inside of the it will be displayed either. This is shown when clicked in the demo as the container shown by the orange border disappears along with the boxes inside.

Conclusion

In conclusion, CSS offers numerous ways to direct the layout of a webpage as you like. Properties such as display, position, and the margin and padding properties are some of the most influential and fundamental properties needed to create a visually appealing layout. The margin and padding properties are great for adjusting how an object or a container sits on the page, giving it extra space or even centering a container on the page with margin: auto;. The display properties and values are wonderful for setting the behavior of an object, and how objects are arranged on the page. Lastly, the position property and values allow for precise placement of object in relation to other objects or just the webpage itself. Overall, learning these CSS options for setting the layout of a webpage will definitely help make you a better designer.
